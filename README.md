# react-native-paypal-checkout

paypal checkout sdk for react native

## Installation

```sh
npm install react-native-paypal-checkout
```

## Usage

```js
import { PaypalCheckoutView } from "react-native-paypal-checkout";

// ...

<PaypalCheckoutView color="tomato" />
```
